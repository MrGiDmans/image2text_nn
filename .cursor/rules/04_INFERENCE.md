# Инференс

## Способы генерации подписи

### 1. Greedy
Берём слово с максимальным вероятностям на каждом шаге.

### 2. Beam Search
Берём K лучших путей.

### 3. Sampling
Сэмплирование по softmax.

---

## Стандартный процесс
image → encoder → features → decoder.generate() → caption

---

## Важные детали
- обязательно <SOS> и <EOS>
- паддинги удаляются
- длина генерируемой последовательности ограничена (max_len)
